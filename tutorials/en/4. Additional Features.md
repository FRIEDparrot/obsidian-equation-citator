## 1. PDF Export

You need to run `Make markdown copy to export PDF` to export a markdown to ensure the styles are exported correctly.  

You can specify `citation colors`, `include captions/descriptions` , and `better image spacing` in the `Settings > categorial > pdf export` page.

The cross file citations will be rendered as normal citation grammar. So the file superscripts will be kept correct when exported. 

Note images may not centered after export, so I recommend you to add a [simple snippet](https://www.youtube.com/watch?v=ngcocqUPiE8) to make image centered. 

## 2. Cache Mechanisms 

This plugin uses cache mechanism to improve performance. You can set higher `Cache Update Time` and lower `Cache Clean Time` for large documents if you encounter any lag. 

## 3. Beta Features  

I strongly recommend you **NOT** to use `Cite with inline code block in callout`. Citations will failed to render inside the callout. Since its a built-in problem for links in obsidian, we won't fully support writing citations in callouts (Including not auto-update and not auto export).

### 版本 1.0.0

版本 1.0.0 - 初始发布

🚀 V1.0.0 主要功能包括：

1. 内联引用（支持实时编辑器和阅读模式下渲染）
2. 跨文件引用；连续引用；
3. 公式自动编号。
4. PDF 导出支持（通过导出 Markdown 副本）。
5. 预览仅在实时编辑器模式下可用。

由于发布版本包含未经充分测试的不安全和不稳定代码，我们不会提供此版本的下载链接。

### 版本 1.1.0

版本 1.1.0 - Bug 修复和主要改进。

🐛 已修复的 Bug：

- 公式渲染样式：现在悬停框将自适应扩展以适应公式内容。
- 公式预览窗口现在会调整位置以避免溢出视口。

其他修复（[Pull Request #7298](https://github.com/obsidianmd/obsidian-releases/pull/7298)）：

1. 修复：使用 `app.fileManager.trashFile(file: TAbstractFile)` 替代 `delete()` 来删除文件。
2. 更改颜色设置机制，避免直接用 JavaScript 设置样式。所有 `setProperty()` 现已移除。
3. 所有设置回调函数使用 `async` 以确保设置正确保存。并添加 `Debugger.log` 用于调试。
4. 移除不安全的 `innerHTML` 操作

✨ 新功能：

1. 自动补全功能现已可用！
2. 阅读模式下公式预览现已可用！
